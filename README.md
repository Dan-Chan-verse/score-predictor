
# Student Exam Score Predictor

A Machine Learning web application that predicts student exam scores based on study habits, attendance, sleep patterns, mental health, and lifestyle factors.

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Features
- Real-time score prediction
- Interactive Streamlit dashboard
- Data-driven performance analysis

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
